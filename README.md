# GearLib

Convert mixing database from a software to another.

## Working
- Rekordbox -> VirtualDJ

## Todo
- VirtualDJ -> Rekordbox
- VirtualDJ -> Serato
- VirtualSJ -> Tracktor
- Rekordbox -> Serato
- Rekordbox -> Tracktor
- Serato -> Rekordbox
- Serato -> Tracktor
- Serato -> VirtualDJ
- Tracktor -> Rekordbox
- Tracktor -> VirtualDJ
- Tracktor -> Serato

# Compile

## Windows
```bash
python make.py %cd%\src %cd%\build\main.exe
```

## Unix
```shell
cd ./build
cmake ..
make
```
